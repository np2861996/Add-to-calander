== Description ==

Just install Plugin and enjoy feature of Insert Body Class.

This plugin used for add body class.

1). Create a page like About or any other.

2). add shortcode

3). Save And it's Done.

== Installation ==
1. Upload the entire `add-to-calander` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
